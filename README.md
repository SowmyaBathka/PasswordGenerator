# Password Generator

A Python command-line application that generates strong customizable passwords based on user preferences.

---

## Features

- Generate secure random passwords
- Custom password length
- Multiple password generation
- Include numbers option
- Include special characters option
- Strong password logic
- Input validation
- Exception handling
- Modular code structure

---

## Technologies Used

- Python 3
- random library
- string library
- Git
- GitHub
- VS Code

## How to Run

1. Download or clone the project
2. Open terminal in project folder
3. Run:

```bash
python main.py
```
